# Crutch Design & Structural Analysis (SolidWorks)

## Overview
This project involves the design and analysis of a mobility aid (crutch) using SolidWorks CAD and simulation tools. The goal was to model key components, create an assembly, and evaluate structural performance under load.

## Objectives
- Create 3D models of crutch components
- Generate 2D engineering drawings
- Perform structural simulations (stress, strain, displacement, factor of safety)
- Validate simulation results with manual calculations
- Compare material performance for design optimization

## Tools & Technologies
- SolidWorks (CAD & Simulation)
- Engineering calculations
- Material analysis

## My Contributions
- Designed and modeled key components (e.g., armrest, tee, pins)
- Performed structural simulations on the crutch cap
- Conducted manual calculations for stress, strain, and factor of safety
- Compared material performance (Rubber vs Silicon)
- Assisted in assembly reconstruction and troubleshooting
